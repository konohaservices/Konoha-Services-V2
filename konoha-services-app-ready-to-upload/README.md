# How to Import Your App

Since direct zip uploads aren't supported, please follow these steps to move your code here:

## 1. Package.json
Open your `package.json` file, copy the content, and paste it into the chat. This allows me to install the correct dependencies.

## 2. Environment Variables
If you have a `.env` file, **do not paste secrets** (like API keys) directly.
- Paste the structure (variable names).
- I will help you set them up securely.

## 3. Source Code
Copy and paste your key files. You can paste multiple files in one message using this format:

```typescript
// filename: src/App.tsx
import React from 'react';
... code ...

// filename: src/components/MyComponent.tsx
... code ...
```

## 4. Assets
If you have images, I can help you find public URLs for them or use placeholders.
